Scatter help

What is Scatter?

"Scatter is a browser extension that allows you to sign transactions for multiple blockchains and provide personal information to web applications without ever exposing your keys or filling out forms."

Getting Scatter

Get Scatter from the Chrome Store. Currently only the Chrome web browser is supported.

Accessing Scatter

Access Scatter by clicking the Scatter icon on Chrome's toolbar.

Adding your EOS key pair

  • Click 'Key Pairs' on Scatter's main menu.
  • Click 'New'.
  • Select 'EOS' in the drop down menu.
  • Enter a name for the key pair in the 'Name' box.
  • Enter your private key in the 'Private Key' box.
  • Click 'Save'.

Adding the EOS network

  • Click the cog icon on Scatter's main menu.
  • Click 'Networks'
  • Click 'New'.
  • Select 'EOS' in the drop down menu.
  • Enter a name for the network in the 'Name' box.
  • Select 'http' or 'https' as appropriate from the protocal menu.
  • Enter the domain or IP address in the 'Domain or IP' box.
  • Enter the port in the 'Port' box.
  • Click 'Save'.

Creating your identity

  • Click 'Identities' on Scatter's main menu.
  • Click 'New'.
  • Select the network and key pair from the drop down menus.
  • Click 'Import'.
  • Click 'Save'.
